# 👑 CampusX AI Engineering Tracker (Max Aura Edition)

A highly-addictive, gamified progress tracker built specifically for mastering AI Engineering. Forget boring spreadsheets—this is a **Gen Z-themed Productivity Engine** designed to keep you locked in.

Built with **Astro**, **Tailwind CSS**, and zero backend (100% lightning-fast `localStorage` state management).

---

## ⚡ Features That Go Hard

### 🎯 The Smart Dashboard
*   **Up Next Engine:** Automatically scans your progress across 328 videos and tells you exactly which video you left off on with a one-click "Continue" button.
*   **Pace Calculator:** Enter your target completion date, and the dashboard calculates exactly how many videos you need to watch daily to hit your goal.
*   **Dynamic Quests:** An auto-generating To-Do list that adapts to your actual progress.

### 🎲 The Black Market (Rewards Shop)
*   Earn Gold by completing videos and Pomodoro sessions, then gamble it all on the **"W or L" Gacha Box**.
*   **Drop Rates:**
    *   **50% (Mid):** Small XP bundles.
    *   **30% (Valid):** "Touch Grass" Passes and Double XP Potions.
    *   **15% (Tuff):** Exclusive themes like the *W Rizz Ocean Theme*.
    *   **5% (Bussin):** The Legendary Jackpot (1000 Gold + 5000 XP).
*   **Daily Deals:** A rotating Dark Web market featuring limited-time, overpriced cosmetics.

### 🍅 Gamified Pomodoro Timer
*   A sleek, glowing focus timer that awards XP and Gold for every 25-minute session you survive.
*   **"Ain't Doin Allat" Token:** Pull this legendary token from the Gacha Box and use it to instantly bypass a Pomodoro session, claim the XP, and force a break.

### 💅 Max Aura Cosmetics
Flex your progress by unlocking global UI themes:
*   **Tuff Hacker Aura:** Paints an animated, falling Matrix-rain across the background of the entire app.
*   **Max Aura Sigma Title:** Replaces the standard dashboard title with a massive, glowing, animated rainbow CSS gradient. Peak looksmaxxing.

## 🛠️ Tech Stack
*   **Framework:** Astro 🚀
*   **Styling:** Tailwind CSS (with intense use of glassmorphism, glows, and custom micro-animations).
*   **State Management:** Native `localStorage` (Instant load times, works offline, never loses sync).
*   **Deployment:** Vercel (1-click deploy ready).

## 🚀 Running Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/projectanshhh/campusx-ai-tracker.git
   cd campusx-ai-tracker
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
4. Open `http://localhost:4321` and start grinding.
